# 🚛 Gestão de Custos Pessoal - Projeto Back-End

![Status](https://img.shields.io/badge/Status-Em_Andamento-yellow)

Este repositório hospeda a API de suporte ao sistema de **Gestão de Custos**, um controle financeiro inteligente focado na logística de transporte. Desenvolvido inicialmente para suprir as demandas administrativas de um escritório de caminhoneiro, o projeto é um marco no meu estudo de sistemas que exigem escalabilidade e regras de negócio sólidas.

## 🚀 Foco e Aprendizados

Este projeto funciona como meu laboratório prático para a aplicação de conceitos avançados, onde destaco:

- 🏗️ **Arquitetura de Software**: Foco em construir uma estrutura robusta e escalável, preparada para crescer conforme a complexidade do negócio aumenta.
- 📐 **Padrões de Design**: Aplicação de boas práticas de desenvolvimento e design patterns para garantir um código limpo e de fácil manutenção.
- 🚛 **Regras de Negócio**: Implementação de lógicas complexas voltadas especificamente para o setor de logística e transporte.

## 🛠️ Funcionalidades e Tecnologias

Atualmente, o sistema já conta com o gerenciamento de:

- ⛽ **Postos**: Cadastro detalhado de parceiros e postos de combustível.
- 🛣️ **Abastecimentos**: Registro preciso de consumos e transações realizadas.

### Tecnologias aplicadas:
- **Back-End**: Java com Spring Boot.
- **Banco de Dados**: Foco em persistência eficiente de dados.
